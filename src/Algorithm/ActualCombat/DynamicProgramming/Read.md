# 动态规划问题的特征

- 首要是状态设计；然后是状态转移
- 首先可以切分为一个个子问题，每一个子问题都是一个最优解
- 可以用递归来解决
- 递归函数是一个 状态=子问题=子问题的解决方案
- 递归可以用记忆化搜索来优化
